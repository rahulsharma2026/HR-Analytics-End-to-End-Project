# 📊 HR Analytics — Employee Attrition Analysis
### End-to-End Data Analytics Project | Excel • MySQL • Python • Power BI

---

## 🎯 Project Overview

This project analyzes **1,470 employee records** from IBM's HR Analytics dataset to identify key drivers of employee attrition and provide actionable insights for HR decision-making.

**Key Question:** *Why do employees leave, and who is most at risk?*

---

## 🔍 Key Findings

| Finding | Insight |
|---|---|
| 📌 Overall Attrition Rate | **16.1%** — above industry average |
| 🔴 Highest Risk Dept | **Sales — 20.6%** attrition rate |
| 👶 Most Vulnerable Age | **18–25 years — 34.8%** attrition |
| ⏰ Overtime Impact | Overtime employees leave **3x more** (30.5% vs 10.4%) |
| 💰 Salary Impact | Low income employees have **highest** attrition |
| ✅ Most Stable Group | Age **36–45** — only 9.2% attrition |

## 📊 Analysis Breakdown

### 1️⃣ Excel Analysis
- Removed 3 constant columns (no analytical value)
- Verified zero missing values across all 35 columns
- Built pivot tables: Department-wise, Age Group, Overtime analysis
- Created bar charts for visual comparison

### 2️⃣ MySQL Analysis (10 Queries)
- Overall attrition rate calculation
- Department-wise attrition segmentation
- Age group breakdown
- Overtime vs attrition impact
- Salary range vs attrition correlation
- Job role wise analysis
- Gender analysis
- Tenure group analysis
- Work-life balance vs attrition
- High-risk employee identification

### 3️⃣ Python EDA
- Libraries: Pandas, Matplotlib, Seaborn
- Correlation heatmap
- Attrition distribution visualizations
- Feature analysis across departments, age, salary

### 4️⃣ Power BI Dashboard
- 6 KPIs: Attrition Rate, Avg Salary, Avg Tenure, Overtime %, Gender Split, Dept Risk
- Interactive filters by Department, Age Group, Gender
- Visual breakdown of all key attrition drivers

---

## 💡 Business Recommendations

1. **Reduce Overtime** — Biggest single driver of attrition
2. **Increase Entry-Level Salaries** — Young employees (18–25) leave for better pay
3. **Focus Retention on Sales Dept** — Highest turnover risk
4. **Improve Work-Life Balance Policies** — Rating 1 employees leave most
